Windows 8.1 app develop - Introduction

Windows 8.1 app - Introduction

Develop Win 8.1 app with JavaScript.

  1. 死心吧, 請準備一台 Windows 8.1 的電腦(一直以來我都以為生為前端攻城獅只要用 Mac 就可以打天下了)

  2. 下載 Visual Studio 2013, 要下載 Visual Studio Express 的版本

  3. 開始努力地讀文件 Doc

Notice & Tip

先了解一些限制與規範以免踩雷

  • Microsoft 提供 WinJS 來協助開發 Windows 8.1 app

  • 無法使用網路的 source

    • 無法使用 remote 的 CSS or JavaScript, 必須全部 download 下來

  • 無法直接使用 iframe

    • package.appxmainifest -> 內容 URI, 把 iframe url 設成例外, 但是該 url 必須用 https
    • 使用 x-ms-webview tag 來代替 iframe, x-ms-webview element | x-ms-webview object

Handle AJAX

在開發 Windows 8.1 時, 並不能直接的 XMLHttpRequest 處理 AJAX 行為, 需用 WinJS 來處理 WinJS.xhr function
