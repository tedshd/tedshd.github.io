Apache - Apache Command Line on Mac / Ubuntu

Mac

sudo apachectl start

sudo apachectl restart

sudo apachectl status

sudo apachectl stop

Ubuntu

sudo /etc/init.d/apache2 start

sudo /etc/init.d/apache2 restart

sudo /etc/init.d/apache2 status

sudo /etc/init.d/apache2 stop

sudo service apache2 start

sudo service apache2 restart

sudo service apache2 status

sudo service apache2 stop

Refer - Ubuntu Linux: Start / Restart / Stop Apache Web Server

JavaScript - scrollTop

scrollTop

Element.scrollTop - Web API Interfaces | MDN

Issue in firefox

Normal

element.scrollTop = intValue;

But when I use

document.body.scrollTop = 0;

It doesn’t work in firefox.

Solution

document.documentElement.scrollTop = 0; // for firefox
