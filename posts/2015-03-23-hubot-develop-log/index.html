# Hubot - Develop log

HUBOT

## Develop Environment

  • Mac OS X

## Require

  • nodejs(install from brew)
  • redis(install from brew)

## install 

npm install -g yo generator-hubot

製作資料夾

mkdir myhubot
cd myhubot

設定 hubot

yo hubot

? Owner: <bot owner>
? Bot name: <bot name>
? Description: <description about this bot>
? Bot adapter: <campfire or shell>

資訊會放在 package.json

Refer - Getting Started
