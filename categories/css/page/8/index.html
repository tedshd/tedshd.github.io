CSS - 垂直置中

on 2014-04-07

CSS - 垂直置中

文字垂直置中

之前 ApplyBOY 在網路上分享了一個 CSS 垂直置中的一個解法 這裡整理出個小小模組出來

假如那一個區域內的東西需做垂直置中, 在該區域的 element 加上 class vertical-centering

.vertical-centering:before {
  content: '';
  display: inline-block;
  vertical-align: middle ;
  height: 100%;
}

該區域的文字就會垂直置中

Example

區域垂直置中

如要在一區域內有一小塊區域置中則需把該區域做 vertical-align: middle;

.vertical-centering-area {
  vertical-align: middle;
}

.vertical-centering:before {
  content: '';
  display: inline-block;
  vertical-align: middle ;
  height: 100%;
}

CSS - 文字圍繞陰影

on 2014-01-10

CSS - 文字圍繞陰影

text-shadow

利用 text-shadow 可達到文字陰影的效果, 但如果要讓陰影圍繞著文字就需要一點技巧了

CSS

h1 {
    text-shadow: #ff0000 1px 2px 3px;
    /*-- 顏色 x位移 y位移 模糊度 --*/
}

要達到圍繞的效果, 必須有上、下、左、右、左上、左下、右上、右下, 最少 8 個方向的位移

左上 上 右上
左 文字 右
左下 下 右下 
h1 {
    text-shadow:
        #ff8800 5px 0 1px,
        #ff8800 -5px 0 1px,
        #ff8800 0 5px 1px,
        #ff8800 0 -5px 1px,
        #ff8800 5px 5px 1px,
        #ff8800 -5px -5px 1px,
        #ff8800 5px -5px 1px,
        #ff8800 -5px 5px 1px;
}

codepen

此例可看出尚未完美之處(要把位移對得很精確) text-shadow 為 CSS3 需 IE10 以上才支援 疊太多 shadow 在上面會影響頁面 render 所以此效果儘量少用

text-stroke

CSS3 的 text-stroke 可完美達成, 但目前只支援 Chrome, Sarfari, Opera

h1 {
	color: black;
	-webkit-text-fill-color: #fff;
	-webkit-text-stroke-width: 1px;
	-webkit-text-stroke-color: #ff8800;
}
